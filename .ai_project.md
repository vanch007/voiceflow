# VoiceFlow Project

## Project Overview
VoiceFlow is a macOS menu bar app for voice-to-text transcription using local AI models. It captures audio, sends it to a local Python ASR server (Qwen3-ASR on MLX), and injects text into active applications.

## Tech Stack
- **Frontend/Native**: Swift 5.0+, SwiftUI, AppKit
- **Backend**: Python 3.11+ (FastAPI/WebSocket)
- **AI/ML**: MLX (Apple Silicon optimization), Qwen3-ASR
- **Build System**: Xcode (`xcodebuild`), Shell scripts

## Directory Structure
- `VoiceFlow/`: Swift Xcode project root
  - `VoiceFlow.xcodeproj`: Xcode project file
  - `Sources/`: Swift source code (`App`, `Core`, `UI`)
  - `Resources/`: App assets
- `server/`: Python backend server
  - `main.py`: WebSocket server
  - `mlx_asr.py`: ASR inference engine
- `scripts/`: Utility scripts
  - `build.sh`: Builds the macOS app using xcodebuild
  - `setup.sh`: Sets up Python environment
- `VoiceFlow.app`: Compiled application bundle (output)

## Current Status
- **Core Features**: Audio capture, ASR transcription, Text injection, Hotkey management
- **Build Status**: Compiles via `scripts/build.sh`
- **Known Issues**: Permissions (Accessibility/Microphone) need reset after rebuild
